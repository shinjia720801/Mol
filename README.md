# Mol

Python 學習與資料分析專案，涵蓋股票資料處理、資料分析與機器學習主題。

## 專案結構

```
Mol/
└── 2026/
    ├── First_Class/                  # 第一門課：課程筆記（依週次整理）
    │   ├── Week_1/                    # 第一週
    │   ├── Week_2/                    # 第二週
    │   ├── Week_3/                    # 第三週
    │   └── Week_4/                    # 第四週
    └── Second_Class/                 # 第二門課
        └── Machine_Learning/         # 機器學習主題
```

## 環境

- Python 3.x
- Jupyter Notebook

## 說明

- `2026/First_Class/`：第一門課的課程 Notebook，依 `Week_N` 週次分類。
- `2026/Second_Class/Machine_Learning/`：第二門課，機器學習相關 Notebook。
